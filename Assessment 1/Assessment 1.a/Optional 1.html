HTML tags and CSS properties quick-reference guide

HTML tags

1. <html> and </html>

Number 1 on our list is <html>. <Html> is always place at the beginning that indicate that
the webpage is html code. And its always followed by a syntax code </html>. </html> is always the last element.

Example

<!doctype html>
<html>
<head> 
</head>
<body>
</body>
</html>

2. <head> and </head>

The header section of an HTML text. Furthermore, this is the place where you will find metadata, which includes
information about the document's styling, the JavaScript libraries used, the title, and the CSS files that were used.

Example

<!doctype html>
<html>
<head> 
</head>
<body>
</body>
</html>

3. <body> and </body>

The body for an HTML document. In actuality, this is an essential component for every HTML documents, however it is only
a tag that indicates what is shown on the page along with where the most important content is.

Example

<!doctype html>
<html>
<head> 
</head>
<body>
</body>
</html>

4. <br>

Using this fundamental HTML tag, we tell the browser that we want empty lines as well as gaps in the text. In reality, incorporating
some breaks allows our content to breathe, making it simpler for readers and comprehend.

Example

 This is a sentence <br> blah blah blah

                    
Result: This is a sentence
    blah blah blah

5. <p> and <p>

The paragraph tag organizes content into paragraphs. Thus, its aim is to divide a section of text or paragraph,
making it easier to read and ordered.

Example


<p>This is a paragraph<p>

6. <ul> and </ul>

An unordered list simply indicates that there will be bullet points for every thing on the list.
Each item in the list will be represented by a bullet.

Example

Country:
<ul>
      <li> philippines
<li> japan
<li> china
<ul>

Result:
    Country:
          *philippines
    *japan
    *china

7. <ol> and <ol>

The list is ordered. Also includes

  • List Items, that will be properly numbered. As a result, adding more list
    items is quite handy because the digits remain in the right sequence.

    Example

    Country:
<ol>
      <li> philippines
<li> japan
<li> china
<ol>

    Result:
    Country:
          1.philippines
    2.japan
    3.china

    8. Images <img>

    The tag allows users to include images and visuals in the content of any webpage. Linking it in src element
    allows users to define the image's location, while using the alt attribute allows you to present
    an image a title which websites like Google can recognize.

    Example

    <img src="https://images.wwe.com/photo-738260184782902-12g3940j23821k>

    9. Style

    <style> tag is always located inside the <head> tag. You can always change the
    style content with the Color, Font size ,etc.

    Example

    <p style="color:blue;font-size:300px">Yeeehawww </p>

    10. Hyperlinks

    <a> tag is important because this tag allowed you to insert a clickable hyperlink to you own page.
    For example the link of any webpage you want it can connect it to your webpage.

    Example

        <a href="https://www.wwe.com/en-us">Visit us on WWE site </a>

    8 CSS properties

    1. Background

    Background is set to use a common color name

    Example

    <p style="background:pink;">Example</p>

    2. Color

    By using this properties you can change the font color.

    Example

    <p style="color:blue;">color font</p>

    3. Font family

    Font family is where you can change the font.

    Example

    <p style="font-family: Courier, 'Times New Roman'; background: blue;">Different font</p>

    4. Height and Width

    With this properties you can change the height and width.

    Example

    <p style="height: 30px; background: blue;">Height and Width</p>

    5. Padding

    Padding refers to the measure of gap between an element's boundary and the
    content or other information included within it.

    Example

    <p style="padding: 50px; background: blue;">Padding example</p>

    6. Margin

    Marging refers the measurement of space between the content and its surroundings.

    Example

    <p style="margin: 30px; background: blue;">Margin example</p>

    7. Text-align

    This is properties allows you to change the alignment of the element.

    Example

    <p style="text-align: right;">Text-align example</p>

    8. Border

    The border can set the color, size, and the element.

    Example

    <p style="border: 14px black solid;">Border example</p>
